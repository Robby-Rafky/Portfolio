Audio-Visual Object Event Tracking in Time and Space

Project Link

My university dissertation project using openCV python. The project aims to map an audio source to the location of a tracked object effectively simulating the tracked object as being the audio emitter.

The object is tracked through colour thresholding to create a mask, noise reduction to reduce potential interference. Once the object is locked on the program starts to look at a local area centered on the tracked object, this reduces the likelihood that the object is lost.

The program is able to relatively accurately estimate 3D depth data from the 2D frames, and works with any camera lens given the parameters are set for it.

Path of Exile Market Lookup Tool

Project Link

A personal project to improve data manipulation techniques with the pandas library and UI skills.

Data from the POE.ninja api is used to display relevant market data in the game Path of Exile

POE.ninja api contains different JSON datasets each with slightly differing formats for different classes of items.

The multiple JSON files are combined into one large file with irrelevant data trimmed off, differing formats reformatted to fit one unified format and various anomalies are accounted for, such as items having variants.

The data is loaded into a pandas dataframe which is then used with tkinter to display all the data.

Ponder LiveTrack

Project Link

A larger university group project, a remote water quality monitor to support the UK's Clean Growth Grand Challenge.

The device transmits data collected from autonomous measuring of water quality (temperature, TDS, turbidity and pH), the data is transmitted via GSM/GPRS or bluetooth (when closer) to an android application called PONDER Livetrack. The mobile application was my contribution to the project.

The application was written in Kotlin using Android Studio, written for Android 10 devices, with support for system light/dark mode.

The app is capable of displaying live data, while updating minimum and maximum levels for each datastream, displaying up-to date gps information with a link to open the default maps app on the device.

The app doubles up as a remote for the device allowing you to turn it off and on when needed and start/stop sessions, also reporting on the device's current status.

The session data is recorded and saved locally within a text file that can be opened and read within the application, everything being timestamped for future data analysis.

Games Match-Maker Discord Bot

Project Link

A personal project, a discord bot that attempts to sort players into fair teams based on match-making-rating (Skill level) while also accounting for what roles the individual players want.

The players all use discord's reactions on a message to select what roles they're interested in playing, then the bot tries to optimise two teams with 5 players each.

The bot will always prioritise placing players within the roles they want to play before trying to optimise roles. In the case where players aren't able to be sorted, the bot suggests what would allow for sorting to work (x player should choose more roles, or x role needs more players).

Older Projects

  • Matlab Image Search Algorithm

    A university project using MatLab to run an image search given an input image.
    Sections of all the images in the test dataset are measured for their rgb composition and those values are represented with an eigenvector.

    The image that being searched goes through the same process, having its own eigenvector and by sorting the eigenvalues when comparing the input image and image dataset, the program is able to return a bunch of similar images to the input, ranking them in order of most to least similar.

  • Projectile Physics Simulation

    One of the first Python projects I've worked on.

    The project uses wx-widgets and visual-graph on Python 2.6 to simulate the flight path of a projectile thrown from a starting position, and it takes multiple different parameters for the simulation:

    Wind, this parameter allows the program to simulate wind at a specific strength and angle.

    Coefficient of Restitution, this parameter allows the program to simulate the projectile as being a bouncy object, bouncing a number of times with return heights defined by coefficient.

    Initial parameters can be set for the projectile such as velocity and angle.

    The program displays this simulation in a 3D space that can the user can explore and logs the whole simulation so the user can scrub through the whole simulation via a scrollbar to pinpoint certain times in the simulation.

    A very old project that uses various numerical methods to iteratively calculate the roots of functions.

    The methods it uses are: Bisection method, Iteration method and the Newton-Raphson method. The program is also capable of comparing these 3 methods so analysis can be done to measure the speed these various algorithms take to calculate the root.
