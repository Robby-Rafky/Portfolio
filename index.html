Project Link

Audio-Visual Object Event Tracking in Time and Space

My university dissertation project using openCV python. The project aims to map an audio source to the location of a tracked object effectively simulating the tracked object as being the audio emitter.

The object is tracked through colour thresholding to create a mask, noise reduction to reduce potential interference. Once the object is locked on the program starts to look at a local area centered on the tracked object, this reduces the likelihood that the object is lost.

The program is able to relatively accurately estimate 3D depth data from the 2D frames, and works with any camera lens given the parameters are set for it.
